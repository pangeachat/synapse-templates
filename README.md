# synapse-templates

Templates for synapse.

Must mirror the [provided templates from Synapse](https://github.com/element-hq/synapse/tree/develop/synapse/res/templates).
See the [documentation](https://element-hq.github.io/synapse/latest/templates.html) for available
template names and template variables.

The ansible playbook places this repository in the proper place. All templates should
go in the `templates` directory.
